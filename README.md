# Verónica – Voice Assistant Prototype

## 🇺🇸 English

### About the Project
Verónica is an early prototype of a voice-based assistant built in Python.

This project originally started around four years ago, before modern conversational AI systems became widely available. At that time, it was a personal experiment to connect audio input, speech-to-text, language models, and text-to-speech into a single workflow.

Today, the project has been revisited and updated using current tools and APIs, resulting in a much smoother and more responsive experience.

### Features
- Audio recording via microphone
- Speech-to-text transcription (Whisper)
- Conversational AI responses (OpenAI)
- Text-to-speech output (gTTS)
- Context-aware conversation (chat history)

### Tech Stack
- Python
- PyAudio
- Whisper
- OpenAI API
- gTTS

### Project Vision
This is not just a finished tool, but a foundation for something bigger.

Planned improvements include:
- Real-time conversation (lower latency)
- Camera integration and visual recognition
- Identity and personality development (Verónica as a character)
- Running the assistant on external infrastructure (cloud or remote compute)
- More advanced memory and contextual awareness

### Why This Project Matters
This project reflects an early attempt to build a voice assistant before AI reached its current level. Revisiting and upgrading it today highlights how much the technology has evolved — and how ideas can grow over time.

## 🇪🇸 Español

### Sobre el Proyecto
Verónica es un prototipo temprano de asistente por voz desarrollado en Python.

Este proyecto comenzó hace aproximadamente cuatro años, cuando la inteligencia artificial conversacional aún no estaba tan desarrollada. En ese momento, fue un experimento personal para integrar entrada de audio, transcripción de voz, modelos de lenguaje y salida por voz en un mismo flujo.

Hoy, el proyecto fue retomado y actualizado con herramientas actuales, logrando una experiencia mucho más fluida y rápida.

### Funcionalidades
- Grabación de audio desde el micrófono
- Transcripción de voz a texto (Whisper)
- Respuestas conversacionales con IA (OpenAI)
- Conversión de texto a voz (gTTS)
- Conversación con contexto (historial)

### Tecnologías
- Python
- PyAudio
- Whisper
- OpenAI API
- gTTS

### Visión del Proyecto
Este proyecto no es solo una herramienta terminada, sino una base para algo más grande.

Mejoras planeadas:
- Conversación en tiempo real (menor latencia)
- Integración con cámara y reconocimiento visual
- Desarrollo de identidad y personalidad (Verónica como asistente)
- Ejecución en infraestructura externa (cloud o servidores remotos)
- Mejor memoria y comprensión del contexto

### Por qué este proyecto es importante
Este proyecto representa un intento temprano de crear un asistente por voz antes del auge actual de la IA. Retomarlo y actualizarlo hoy muestra cuánto evolucionó la tecnología — y cómo una idea puede crecer con el tiempo.
